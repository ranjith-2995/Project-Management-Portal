# Project-Management-Portal
A full-stack Project Management Portal for creating, managing, and tracking tasks.

Built using:
    Frontend: React.js + Vite
    Backend: Node.js + Express.js
    Database: MongoDB

The application allows users to create tasks, update task status, filter tasks, and manage project activities through a responsive dashboard.
